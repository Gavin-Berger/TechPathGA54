# TechPath GA

TechPath GA is a React Native / Expo mobile app that provides Georgia-specific technology career guidance using an LLM.

## Features

- Professional one-screen mobile UI
- Georgia-focused tech career advice
- Profile selection by experience level
- Interest area selection
- Quick prompt chips
- Fallback response for out-of-scope topics

## Tech Stack

- React Native
- Expo
- TypeScript
- OpenAI API

## Setup

1. Install dependencies:
   npm install

2. Install picker:
   npx expo install @react-native-picker/picker

3. Add environment variable:
   EXPO_PUBLIC_OPENAI_API_KEY=your_key_here

4. Run:
   npx expo start

## Example Questions

- How do I become a cybersecurity analyst in Georgia?
- What certifications are valued in Atlanta tech jobs?
- What Georgia schools are best for software development?
